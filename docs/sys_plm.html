[ Top ] [ sys ] [ Functions ]

NAME

   get_command_tail -- return the pointer to the command tail

SYNOPSIS

   call get_command_tail(p$str, len);

DESCRIPTION

   Write the command tail into the area pointed by p$str of maximum length
   len.  The written string is null-terminated.
